记录一下我的一些剧本灵感

发表于 2024-05-23
  • aurora
首页

我是个喜欢胡思乱想的人，然而我的写作能力有限、不能写出好的故事。这里随便写写一些构想。

我喜欢的题材，是 “绝望中的美好”: 故事背景很残酷、很黑暗；在这种背景下出现一丝丝美好、再将它残忍地摧毁。大概就是这种。然而我玩的 gal 作品中、这种题材比较少，希望有人能给我推荐这类题材的。

我印象里比较深刻地，一是《夏空彼方》，无论怎么样也不能得到十全十美的结局，玩起来有种无力感；二是《杀恋悠久流传的恋之歌》，兄妹刀子题材挺让我触动的。三是《星之终途》，属于是我喜欢的 “绝望中的美好” 的题材了。

兄妹的故事

我文笔不够、构思不紧密，写起来还是很羞涩的。我曾经构思过一个故事，希望能给人留下灵感吧。我没想人物的名字，这里就随便编造，当乐子看看。

在 X 国有个核心的财阀家族，A 家族。A 家拥有着很强大的资本力量，然而内部极其腐朽、各怀鬼胎。到家主 A1 (年龄约莫 50 多岁) 这代，她与忍者的女儿 B1 通婚，生下了哥哥 A2 和妹妹 B2，然而家族内部不允许这种事情发生，A1 对 B1 感到些许愧疚心理，接过了妹妹 B2 的抚养权，把哥哥 A2 留给了 B1。

B2 四岁时、被拐卖到邪恶组织，经过了层层血腥残酷的筛选活了下来，六岁时阴差阳错被带到 A 家族、做妹妹 A2 的保镖、和妹妹上学。

在 A 家族，家主 A1 有三个儿子 (30 来岁)，分别是 A3、A4、A5，然而他们各怀鬼胎、根本没把 B2 当妹妹看。A3 通过手段软禁 A1、成为了家族的实权统治者。B2 缺乏亲情、把 A2 当成亲哥哥黏着 (实际上确实是亲哥哥)。

然而 A2 是忍者的孩子，开写轮眼了 (胡编乱造 Ing)，A2 11 岁时、被邪恶的政治派 C 的统领人 C1 盯上，C1 告诉了 A3，于是 A 家族和 C 政党联合迫害 A2，C1 派得力手下 C2、劫持 A2 的青梅竹马诱导 A2 来到悬崖救人、把青梅竹马当场杀掉、拿到 A2 的写轮眼一只 (继续胡编乱造)，A2 用力量伪装自己死亡成功逃脱、然而体力不支失血过多昏倒在街上。

有一位绝世剑客，D 派系的首领 D1、见到 A2 将其救回、并给他移植了一只眼睛，将 A2 培养成强大的杀手、战斗小队队长，执行各种高难度的任务。A2 见证了无数并肩作战队友的死亡、成为了表面上感情淡薄的冷血杀手，早早开启万花筒写轮眼 (我继续瞎写)。

D1 同样抚养着一个被抛弃的女孩子 D2，比 A2 小三岁，实际上是个狐狸生的孩子，二人以兄妹相称。A2 13 岁时，随着 C 派系的领头人 C1 野心和实力越来越大，某天 C 对 D 派进行悄无声息的进攻，D1 让 A2 带着 D2 逃走、自己战死。然而 C1 的权力很大、行动也很缜密，控制舆论称：A2 杀死 D1 造成灭门惨案、并带着 D2 出逃、给 D2 进行洗脑，于是 A2 遭到全面通缉。

A2 不得已、揽下了所有的罪名、把妹妹 D2 交给了值得信任的、有一方权势的 E 派首领 E1。加入了当时的恐怖组织 X（类似于火影忍者里的晓组织）

过了四年，C派成为了全国最强大的派系、C1通过暗杀对手等方式，成功成为了国家领导人。A家族势力渐渐薄弱、家族陷入十分大的危机。C1的得力手下C3找到A3，将会为他们家族提供免费援助，但条件是B2必须要成为自己的小老婆。此时B2 16岁、C3是40多岁的中年大叔，B2不愿意、遭到A3怒声呵斥、扇耳光。

B2 感觉很委屈：以前受欺负时，总有 A2 来保护、帮助他，现在身边经孤立无援。

另一方面，C 对 X 进行严厉的打击、A2 为了掩护 X 留下来作战、对手是当年陷害自己的 C2，C2 被打败了、临死前嘲讽 A2: 你的妹妹 B2 马上成为别人发泄 xing 欲的玩具了，你什么都做不到、连妹妹都保护不了。

A2始终没有忘掉自己的妹妹B2，赶回去准备救B2，他得知背后的势力很强大、然而还是要保护妹妹。B2在婚礼前被A3强制送到C3家里、C3欲对B2行不轨之事,此时A2出现杀掉了C3，带着妹妹B2逃离。

B2 有这么多年的委屈、无人倾诉、以为哥哥死掉了，再次见到 A2 大哭一场排泄心中的苦闷。

此时社会出现了各种对 C1 的批评的声音，指责他谋权篡位、纸包不住火。C1 忙于各种镇压工作、E1 此时要竞选新的国家领导人，A2 决定帮助、保护 E1，A2 与组织 X 一名叛逃成员取得联系，二人合伙用武力杀掉了 C1。E1 方面保护了 A2 的身份、给他假的身份信息让他生活下去。

C1 垮台后、社会开始进行平反工作，A 家族因为站在了 C 的派系、做过不少脏事、A3 进监狱了。A2 找到被软禁在医院的 A1，帮助 A1 重回家族领导者。A1 知道那几个儿子个个心怀鬼胎、把部分权力给了 A2，A2 把权力给了妹妹 B2。

A2 视力急剧下降、去医院里检查，得知自己患上了不治之症、活不过一年。然而妹妹 B2 却喜欢上了哥哥 A2、极度兄控。在夜里向哥哥表白、想推倒哥哥)，A2 拒绝了，和妹妹进行思想交流。

兄妹二人过着幸福的生活，然而这件事被X组织成员X2知道，X2诱骗B2,告诉B2:A2是个靠杀人为生的坏蛋、很多事情都瞒着你,他现在快死了，救你是想拿你续命，证据是他眼睛不行了、还瞒着你、骗你。

B2 回到家、拿者杯子说是碗、测试哥哥，发现如 X2 所说，B2 哭着跑出了家，A2 去追。遇到了 X2 和一群执法人员，X2 告诉 A2, 自己的目的是利用妹妹 B2 获得能力，A2 拖着疲惫的身躯进行战斗、将 X2 封印了起来、并再次逃脱假死骗过众人，然而因为绝症的原因，在路上昏倒了。
