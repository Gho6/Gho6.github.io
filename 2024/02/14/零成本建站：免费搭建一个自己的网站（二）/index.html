零成本建站：免费搭建一个自己的网站（二）

发表于 2024-02-14
  • aurora
首页

前言：上次简单介绍了 hexo 建站，我自己也实打实的又建了个站：

https://imaginative-tapioca-61dff0.netlify.app/

访问速度应该还可以，用的是 butterfly 主题，托管在 netlify 上，不需要实名（风险小）。网站只花了 1 小时便搭建好，现在已经可以满足一些基本需求了 (比如可以发资源)。不过网站尚未完全完成 (进行开发 ing)。编辑器我用的 markdown2，剩下要做的就是继续美化 + 使用 + 域名了。关于域名想不花钱可以嫖个二级域名，为了安全性，建议使用不需要实名认证的域名。

这一次讲 wordpress 建站 (其实是在讲如何白嫖服务器。。)。我之前的网站 “樱韵” 就是 wordpress 站点。

什么是wordpress？

WordPress 是基于 PHP 和 MySQL 的免费开源内容管理系统（CMS）。 它是全球使用最广泛的 CMS 软件，截至 2019 年 5 月，它为排名前 1000 万个网站中提供了超过 30％的支持，并拥有在使用 CMS 构建的所有网站中，估计有 60％的市场份额。

简单总结：很多很多网站都是用 wordpress 建的，它是个建站程序，功能强大、插件丰富、使用方便。就和 hexo 做一下对比，hexo 要实现搜索评论功能，需要各种配置、很麻烦，如果要实现注册、登录、签到领积分、查看隐藏内容，就更加困难了；而 wordpress 直接就可以实现这些，不需要配置，功能强大，wordpress 有着数不胜数的插件、美化起来一点也不逊于 hexo。

开始白嫖

目前我所知道的白嫖方案：腾讯的在线云平台 cloud studio；白嫖谷歌、阿里云国际版一年服务器；1 块钱（四舍五入为 0）+ 定时延期白嫖阿贝云、三丰云服务器和虚拟主机；国内各大厂商试用；阿里云 “高校计划” 白嫖一年 + 七个月的服务器 (需要学生认证)…

我当时用的阿里云 “高校计划” 给我的 300 块代金券，买了一年的 2 核 1GB+30Mbps + 每月 1TB 流量的新加坡轻量云服务器 (开销 288 块)，后来试了 bbr 加速、配置 cloudflare、网上大神分享的加速脚本，但是效果都不好 (愤怒的我关闭了网站，我亲自试坑，不建议买这个)。

另外一些建议：如果你的网站有些不太合适的东西，不要买国内服务器、需要实名的服务器，否则可能去喝茶。

安装和使用

虚拟主机：直接上传建站程序、配置域名 (域名想白嫖可以用一些厂商分配的二级域名)、解析域名 —— 完毕！具体可以参考百度、b 站等。

云服务器：建议百度和 b 站搜索，这里不细讲了。

备案：国内服务器和主机必须备案（除去港澳台），国外的可以备案也可以不备案。如何备案请百度。

SSL 证书分发：阿里云白嫖、腾讯云白嫖、站长工具白嫖、一些厂商直接帮你申请了就不用考虑了。

接下来？

接下来如何使用 wordpress 就需要你自己学习了 (wordpress 的使用 + 插件用法，写的话几百篇都写不完)

一些推荐的插件

1.wp rocket: 虽然是付费插件（但是有破解版），用于加快网站访问速度

2.elementor：虽然是付费插件（但是有破解版），用来美化网站

3.All in one WP Migration: 虽然有些功能需要付费（但我们可以改代码破解，教程我写过），用于网站备份
