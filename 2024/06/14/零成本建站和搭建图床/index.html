零成本建站的教程我写过了，可前往这串链接查看:https://pd.qq.com/s/6440142m1

需要补充的是，在 “提升速度，托管到 netlify” 前的一步，需要开启 github pages,github 点开上方的 Settings, 点击侧边栏的 Pages 进行部署，网上教程很多，看不懂可以去搜。

如何搭建免费图床?

可以参考B站视频BV13N411z7Lu，搭建完还挺好用。telegramf地址:https://github.com/cf-pages/Telegraph-Image

只要按视频操作就行了，另外有一点，如果你 github 打不开，通通把 github.com 换成 bgithub.xyz 这是镜像站，很快。
