# 关于本博客的介绍

因为搭建 QQ 机器人白嫖了阿里云的服务器，但觉得只用来搭建机器人太可惜了，于是突发奇想搭建了个博客（虽说最后没用到罢了），因为自己在计算机技术上是新人小白，写一些技术类文章基本是不可能的，于是就写写杂谈日记之类的文章，或者写一些视觉小说简评，或许还会分享一些游戏资源。

# 关于自己

一名 galgame 爱好者和东方 project 萌新，计算机小白，宇华。在互联网上冲浪并加入同好的圈子们，喜欢在互联网上收集各种自认为有价值的信息并进行整理，惧怕并反对网暴的网民。

# 关于 galgame

一名 gal 新人，入坑不到一年。第一部玩过的 gal 作品是《美少女万华镜：勿忘草与永远的少女》，只是过了过 CG，后来又重新过了一遍。后来因为同学发给自己的改版《Teaching Feeling》中的广告加了一些群，通过群聊获得了《亚托莉：我挚爱的时光》（明年动画播出）真正入坑。
很多人总是把 gal 乃至整个视觉小说当作 h 游戏，不可否认的是，很多 gal 作品中都有 h 场景（比如比较有名的《缘之空》）。但 galgame 并不是只以 h 为卖点，而且 gal 中也有很多优秀的全年龄类作品（没有 h 的）。gal 中不乏一些优秀的作品：比如《Clannad》《命运石之门》《fate》《日在校园》《白色相簿》《纯白交响曲》……

# 于东方 project

因为听了东方的《bad apple!!》, 从红魔乡入坑，入坑不到半年，没什么发言权。。
