# 公告

在经历了深刻思想斗争后，我最终还是复读去了。一个月回家一次，基本是不会写博客了。意外的发现还会有人看，就写一篇关于找资源的途径的文章。

# 前言

见很多人求资源，有被骗的，又找不到的，所以就写一写自己的一些方法吧。方法不是很好，希望大佬们能给出见解。

到现在 gitee 实名认证没下来，eu.org 域名也没下来，所以访问速度可能会慢些。自己的个人主页：gho6.github.io/aurora6.github.io, 访问速度比较慢。没钱租服务器，也嫌开导航站麻烦不好整，所以合集还是文档版本。

自己也比较怕事，不想在互联网上和人发生矛盾，自认为始终以较谦卑的态度行事，但不排除自己有做错的、损害他人利益等不对的事。。如有不当行为请联系。

# 正文

文章为新人向，旨在帮助更多新人学习自己找资源，主要是 galgame, 让他们少走弯路，大佬请勿喷。如果你有好的见解，欢迎提出。如有不当行为请联系删除。

自己的 gal 交流群：376757863，用来交流 gal 用。

推荐配合文档使用（参考文章 网站投稿分享及网站合集），文档也自认为写得比较详细了，这里再写写说明

# 一.galgame 游戏

首先，谈谈我自己的思路吧。我会对 galgame 年代进行简单地分类：新作、近作、中作、旧作、超古老作品。新作就是新出的作品，近作就是 2008 年至 2022 年的作品，旧作是 08 年之前、00 年后的作品，超古老就是上世纪的作品了。

新作：

入正：首先可以通过 getchu、东京图书馆、Eroge、Media Clip、等平台，或者是游戏官网、各大搜索引擎获得新作信息和发售渠道，知道发售渠道就可以直接去入手正版来玩。一些正版网站了解一下：Steam、DLsite、DMM、Getchu。实在不行直接百度、萌娘百科找入正渠道（有的 gal 发售渠道就比较偏，比如 16 年的《天津罪》（虽然不是新作），推荐萌娘百科一下）

但是很多人可能没有足够的资本去入正，就可以考虑其他途径。首先在国内的环境下，大概就是绯月、2dj、南 +、黑丸这几个论坛，绯月和 2dj 主要是用户自己入正购买后分享出来，南 + 和黑丸主要是 SLG 和 H 游戏比较多，一般用来找绅士游戏，但也能找 gal。

还有一个地方，名叫 bilibili，有的时候 bilibili 总是能给你惊喜。
另外还有 galgame 吧，G 吧是很强大的

但是有的游戏在国内就是找不到，或者不够快，这时候可以把目光转向国外：Anime-sharing—— 算是继承了曾经欧美论坛 hongfire 的遗址了，缺点是分享者总是用网赚盘，一个游戏能下几天（还不如国内等）。

近作和中作：

这些作品一般在国内得到广泛传播，一般可以按照 “失落小站 —— 青桔网 ——acgngame—— 樱落之时 —— 极点 ACG” 的顺序，如果是 18 年之前的作品我还推荐 “忧郁的弟弟”，虽说站长已经基本不更了，但 18 年的作品确实很全。都找不到，去 “花火学园”“绯月”“终点论坛” 等看看，还有万能的 b 站，基本都能找到。

旧作和超古老作品

旧作以上站点同样适用，只不过一些年老的站点也可以用来寻找旧作。一些站点曾经人数很多，现在比较冷清吧，比如 飞雪之城、澄空（澄空解包好评！）、心跳回忆网络（专门针对心跳回忆的）…… 这些站点也可以去看看。

难找的冷门资源

有的东西，可能只有一串日文名连中文名都没，或者只有个图…… 还十分冷门。只有串日文名，最恶心的是同人作品，而且有时你甚至会发现甚至制作方都解散跑路了。。考虑使用 vndb、谷歌、yadex（俄罗斯人不会让你失望）、yahoo 获取尽可能多的信息；然后去 Anime-sharing、绯月、ggbase、nyaa 等碰碰运气，基本能碰上。但如果还是找不到呢？想办法了解该作品的信息，有的作品就是自制出来在小小的同好论坛发布的，搜索基本不可能。在论坛、网络上发求助帖子。实在不行在国内外论坛上求助… 对于图片之类的，找同好识别或者交给百度识图、谷歌识图，总能识别出来的

# 二.gal 补丁

制作：

汉化补丁制作起来可能有困难，因为需要学习程序、日语知识（据我了解大部分汉化组要求日语要过 N2 的）、修图等知识，但是只是机翻一下游戏供自己玩还是可以的，而且现在还有 ChatGPT 这个工具，还是可以自己试试的。

程序部分可以先从解包开始学习，教程推荐 B 站 Dir 大佬的视频、澄空解包，总之善用搜索引擎即可。程序部分也可以在 b 站找，还有 github，估计专门学习计算机的大佬们对这个了解比我多。不要急，慢慢学起。

白嫖：

好吧，我认为大部分人是不会去自己做补丁的，确实有难度，而且比较耗时耗力。这里推荐 Galgame 吧和绯月论坛，大部分补丁都是在这里最先发布的 —— 当然有的汉化组会自己建立网站发补丁，比如破琵琶和萌你妹。还有一个论坛名叫 KDays, 里边入驻的汉化组挺多的，可以去看看。包括万能的 b 站，也可以瞅瞅。

# 三.gal 攻略

b 站，相应的贴吧，城也的部屋… 还有一个被大中快遗忘的站点 —— 恋爱游戏网。虽说恋网倒了，但它的攻略还是可以的。

# 四。单机游戏

参考合集里的部分，推荐 3dm、switch520、海盗湾 等

# 五。站点失效？

善用百度、谷歌等搜索引擎，及时保留官方联系方式，发送邮件、加入 TG/Q 群…

# 六。其余部分

推荐看合集里的 “导航”，那些网站分类比我细。或者在合集里找找，自己看的比较少，但是如果真想找的话，总能找到的。

# 结尾

随便写写，主要是 galgame。希望能方便他人。只是自己的方法，不一定好。但本意是好的。希望大佬们可以分享自己的途径。
