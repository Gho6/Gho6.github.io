css选择器的学习笔记

发表于 2023-12-18
  • aurora
首页

关于选择器，很多人会想到 span、class、#、.、等一大堆杂乱的字符，刚开始还能学会，后来可能就忘了。

于是作为一名解剖一点儿也不会的医学生，写一写自己的学习笔记

选择器大致分为五类，全局选择器、元素选择器（span是一个元素选择器）、类选择器、id选择器、内联选择器。执行的优先级：内联>id>类>元素>全局

  1. 全局选择器

    全局选择器格式为*{}，对于全局生效，一般用于更改全局样式等。

  2. 元素选择器

    元素选择器格式,诸如h1{},h1,h2{},作用是对某个标签的东西生效，相当于全局中“独树一帜”，特殊群体，可以摆脱全局的限制，优先级高。

  3. 类选择器

    可以在元素标签中插入，诸如

    我是笨比

    ，这相当于设置了一个类名为“woshibenbi”，类的名字是有限制的，英文字符、数字、短横线。还可以把“我是笨比”这句话划为两类、三类，只要第一个class标签后加空格，再写其他标签就好了。最后写法为：.woshibenbi{} (一定要用.号)。如果你想把css写的看起来高级冗杂些，还可以用合并选择器+类选择器，结果是网页变得更卡了......

  4. id选择器

    针对一个、一句话，class是针对一类，但是你想在这一类中找个特殊的，id选择器则是针对这一特殊的，给它标个id注明是什么,比如id标textl,就写：#textl{}

  5. 内联选择器

    这个的作用，可以直接选择一段话、一个字，打个比方： 元素选择器是针对国家、类选择器针对民族、id选择器针对某一个村、那么内联选择器则是具体到人，直接句子里插入就行，缺点也很明显，繁杂冗长

    span标签和内联选择器、id选择器、类选择器？

    span标签可以适用好多个个体，给张三李四王五都打上，一个span标签就可以把他们放在一起。那么类选择器不也是可以吗？很遗憾，加入一段话中有连续的三个字在一起要标为特殊，类选择器办不到；而内联选择器只能一个个针对，太麻烦：你叫一个班的人，难道要一个个点名叫出来吗？

    而 id 选择器可以针对一个特殊的语句，相当于挨家挨户叫，内联选择器是一个个叫，元素选择器则是把带有某类标签的全部句子叫出来，类选择器则是分类：想怎么分怎么分，就算是 h5 和 p 标签两种完全不同的，也可以划为一类。
