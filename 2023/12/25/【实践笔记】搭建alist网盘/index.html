【实践笔记】搭建alist网盘

发表于 2023-12-25
  • aurora
首页

前几天苦于网盘容量不足（度盘容量快没了…），又不想交钱。F 岛站长岛主酱建议我搭建个 alist 网盘。刚听到时，这个东西好像经常见？很多资源站点都用这个挂载资源来着？可以挂载很多网盘还可以把电脑文件映射上去。

于是我先看看手里的垃圾级别云产品：一个白嫖织音云三个月的主机，用来托管网站引导页；还有两个小兔互联虚拟主机，花了 22 块用于搭建论坛（可以当发资源的备用站点）和个人主页；linux 轻量服务器让我用来搭建网站 “樱韵” 了（等待我复活吧 hh）；最后还有个白嫖的学生云服务器用来搭建 QQ 机器人玩。（没钱的穷比学生党，大佬可怜一下吧 QWQ）

经过深思熟虑，网站 樱韵 刚被我折腾坏了… 配置还不好；而学生云服务器就用来运行个机器人，于是就用 windows 系统的学生服务器搭建 alist 网盘。

首先进入 github 下载 alist 最新的版本，找到 amd64.zip，下载到服务器解压；接着 cd 到 exe 的目录，输入 alist server 运行；最后用户名 admin, 密码、端口号会在命令提示符显示，浏览器打开 127.0.0.1: 端口号（我的是 5244），登录即可。

登陆完点击管理，可以改密码，挂载网盘参考 https://alist.nn.ci/zh
看官方文档即可。这里我挂载的度盘，点击存储选项，驱动选择度盘，挂载路径为：/ 你想起的名字 ，web 代理打开，策略为：使用代理地址。接着找到下边的 “刷新令牌” 点我，登录度盘账号，只需复制 refresh_token: 的内容即可，粘贴到 “刷新令牌”，至于客户端 ID 和密钥，我的一开始是配置好的，client_id: 和 client_secret: 的值就是，最后保存。

设置权限

点击用户，guest是游客，可以设置游客权限。如果想添加用户的话也可以加。

最后是如何访问，打开服务器 “安全组” 选项，端口范围填端口号，授权对象 0.0.0.0/0

最后打开浏览器，输入你的服务器 ipv4 地址：端口号 就可以成功访问了！
当然也可以弄个域名解析一下，新人 1 块钱都能买一年域名。

由于我挂载的度盘，下载速度可想而知，不过不光度盘，alist 还可以挂载多种网盘。

整体搭建起来简单，服务器配置要求低、小带宽服务器也可以搭，不走服务器流量，如果你是和我一样的穷比学生党还可以去白嫖厂商服务器，最后留下岛主酱给我的 linux 一键安装脚本：

curl -fsSL "https://alist.nn.ci/v3.sh" | bash -s install

如果你想挂载网盘到我的小站，可以把一些必要信息留给我（如刷新令牌、你想要给目录起的名字之类的）等风头过去再继续

最后偷偷进行网络乞讨，旨在为网站运营谋求一些费用（穷苦学生党）。
