【感想】使用Alist后随便写写

发表于 2023-12-27
  • aurora
首页

前几天搞了个 Alist 网盘，用起来感觉不错，更可喜的是，这个网盘还可以挂载别的 Alist 站的资源，解决了我网盘空间不够的问题。在整理的网站合集加持下，加上万能的 “yandex.com”, 我迅速建立了一个庞大的资源库。

恰巧最近一直在学 html+css，于是终于可以学有所用，用传统的 markdown 语法 + html 将页面稍微美化了一下：除了没有域名外，看起来还挺不错。

为什么选择Alist？

Alist支持20多种网盘，也就是说，你可以把你的百度网盘、阿里云盘、115网盘、蓝奏云等一堆网盘挂载到Alist上，可以随时随地访问（当然有时也会抽风，是个问题）。而对于我这种没有足够空间的穷b，Alist支持挂载别人的Alist网盘资源（前提是别人设置了允许挂载）。简单地说，就是可以把别人的Alist网盘资源全拷贝一份在自己的网盘上。

更重要的，Alist 对服务器配置要求不高，而且流量不走服务器，文件也可以设置密码，对于资源分享者来说是个不错的选择，做到随时随地下载。

搭建一个Alist网站需要多少钱？

如果要把白嫖做到极致的话、一切都很理想的情况下，能0元建站：服务器全靠白嫖，域名也白嫖，没有任何人攻击网站（，我的用的IP地址访问，服务器白嫖阿里的，可以运行半年，花了0元

缺点

Alist的搜索功能，我觉得不太好用，东西多了创建索引耗时长、检索不出来;另外Alist的安全性不好。另外值得注意的是，如果你一些网盘（如百度云）开启了照片备份，然后你又把度盘挂载上去...你的隐私就被当众曝光了
