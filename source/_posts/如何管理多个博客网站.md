这是自己之前碰到过的问题：在github、gitee、netlify都有博客网站，在部署时不知道怎么管理，怕把东西传错了、传不上去。

***

# 解决方法

很简单，第一步，更改全局设置。打开git bash

git config --global user.name "xxxx"

git config --global user.email "xxxx@xxxx.com"

***
第二步，改ssh（我用的ssh，其它的我不管...）
打开C:\Users\用户名\.ssh  换文件就行。
